Diffie-Hellman Key Exchange

A visual simulation of secure key exchange over an insecure channel

1. Public Parameters

These values are agreed upon publicly and known to everyone (including potential eavesdroppers)

Must be a prime number
Must be less than p

2. Private Keys

Each party selects a secret number that never gets transmitted

Alice

Computes: A = ga mod p

Bob

Computes: B = gb mod p

3. Public Key Exchange

Alice and Bob exchange their public keys over the insecure channel

Alice's Public Key (A) A = ga mod p
⟶
⟸
Bob's Public Key (B) B = gb mod p

4. Shared Secret Computation

Both parties compute the same shared secret independently

Alice computes: s = Ba mod p
Bob computes: s = Ab mod p
🔒 Shared Secret Both parties now share this secret!
